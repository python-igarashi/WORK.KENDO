研修会・春の大会・秋の大会の3つの業務を操作するGUIを、Python標準GUIで作成する。

起動するとまず「研修会」「春の大会」「秋の大会」を選択できる。
選択後、その大会の操作ボタンを表示し、押下結果の標準出力を画面下部に表示する。
標準出力エリアはコピー可能で、実行ごとに一度クリアする。
実行中は操作ボタンを無効化し、ステータス表示を「実行中…」にする。
完了後はステータス表示に「大会名 > 操作名」を表示する。
トーナメント作成と春の大会の選手集計は、完了後ステータス表示に
「大会名 > 操作名 > 出力ファイルパス」を表示し、クリックで出力フォルダを開ける。

標準出力エリアには「コピー」ボタンを付け、内容をクリップボードにコピーできるようにする。

トーナメント作成（05_CreateTournament.py）実行時は標準入力が必要なため、GUI側で入力ダイアログを出す。
ダイアログには以下を表示・操作できるようにする。
- 前回の抽選結果番号
- （前回の抽選結果番号の発行日時）
- 選択肢はラジオボタン3種
  - 前回の抽選結果番号を使用
  - 新規抽選（new）
  - 数字を指定（右側に数字入力欄）
数字指定の場合のみ入力欄を有効化する。
キャンセル時は実行しない。

# 研修会
## URLを出力
押下すると以下を実行する
cd 研修会
python 01_PrintURL.py

## ダウンロード
押下すると以下を実行する
cd 研修会
python 02_DownloadTSV.py

## 選手集計
押下すると以下を実行する
cd 研修会
python 03_SumPlayer.py

## (注意)入力シート再作成
押下すると以下を実行する
cd 研修会
