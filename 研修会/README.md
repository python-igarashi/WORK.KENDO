# 研修会 - 使い方

研修会向けのデータ取得・データ出力を行う Python ツール集です。

## 前提
- ルートフォルダのREADMEに従い、事前準備（環境構築）を済ませていることを前提とします。

## セットアップ
- 入手済のGoogle認証ファイルをこのフォルダに配置してください。

## サブフォルダの説明
- `DownloadTSV/`：各団体が記入したGoogleシートが、手順2によってこのフォルダにTSV形式でダウンロードされます。

## 操作手順
1. GoogleシートURLを出力する<br>
 この操作では、各団体が記入を行うGoogleシートのURL、および、集計結果/テンプレート/記入例のGoogleシートURLを出力します。<br>
 出力結果を、研修会参加者の記入を行うURLとして各団体に連絡してください。<br>
 （コマンドプロンプトで以下を入力してリターンキーを押します。）<br>

    python 01_PrintURL.py<br>

2. GoogleシートをTSVファイルフォーマットでダウンロードする<br>
 この操作では、全団体のGoogleシートをTSVファイルフォーマットでダウンロードします。<br>
 ダウンロードしたTSVファイルは、後の操作手順で出力するデータの元となります。<br>
 （コマンドプロンプトで以下を入力してリターンキーを押します。）<br>

    python 02_DownloadTSV.py<br>

3. 参加者を集計する<br>
 この操作では、ダウンロード済のTSVファイルを元に、研修会参加者を集計します。<br>
 結果は、集計結果GoogleシートのURL（操作手順1を参照）に出力されます。<br>
 （コマンドプロンプトで以下を入力してリターンキーを押します。）<br>

    python 03_SumPlayer.py<br>

## 操作手順(要注意)
0. 各団体のGoogleシートを再作成（初期化）する<br>
 この操作は、システム管理者（システムに紐づくGoogleアカウント管理者）のみ実行可能です。<br>
 この操作では、各団体が記入を行うGoogleシートを再作成（初期化）します。通常は、研修会業務中の操作手順1の事前に一度だけ実施します。<br>
 再作成の元となるのは、テンプレートGoogleシート（操作手順1を参照）です。記入フォーマット変更等、必要に応じ、事前にテンプレートGoogleシートを修正しておいてください。<br>
 各団体の記入は全て抹消されるので、操作には十分に注意してください。必要に応じ、操作前に各団体用Googleシートをバックアップするなどの対策を行ってください。<br>
 （コマンドプロンプトで以下を入力してリターンキーを押します。）<br>

    python XX_RemakeFiles.py<br>
